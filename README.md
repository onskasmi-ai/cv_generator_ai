# AI CV Generator

Application web permettant de générer automatiquement un CV ou une lettre de motivation.

## Fonctionnalités

- Génération de CV
- Génération de lettre de motivation
- Français / Anglais
- Export PDF

## Technologies utilisées

- Python
- Flask
- HTML/CSS
- FPDF

## Installation

pip install -r requirements.txt

## Exécution

python app.py
